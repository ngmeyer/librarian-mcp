# Contradiction Protocol

Most research buries contradictions. This system surfaces them on purpose. The tension between conflicting perspectives is where the real insights hide.

## When You Find a Contradiction

### Step 1: Document It Clearly
```
## Contradiction: [Short Name]
**Lens A says:** [finding]
**Lens B says:** [finding]
**Why they disagree:** [different data, different timeframe, different assumptions, different values]
```

### Step 2: Classify It

| Type | Description | Example |
|------|-------------|---------|
| **Data conflict** | Same question, different numbers | GDP growth: IMF says 3.2%, World Bank says 2.8% |
| **Interpretation conflict** | Same data, different conclusions | Both see declining birth rates; one says crisis, other says adaptation |
| **Timeframe conflict** | True at different time scales | Short-term bearish, long-term bullish |
| **Values conflict** | Different priorities lead to different judgments | Economically optimal ≠ socially just |
| **Scope conflict** | True at one scale, false at another | Works for small countries, fails for large ones |

### Step 3: Rate the Tension

- **Critical:** Contradicts a core finding. Must be resolved or prominently flagged.
- **Significant:** Changes the nuance of a finding. Include in synthesis.
- **Minor:** Interesting but doesn't change conclusions. Note for completeness.

### Step 4: Attempt Resolution

Try these in order:
1. **Check sources.** Is one side using weaker evidence? Apply [[source-evaluation]].
2. **Check timeframes.** Are both true at different points in time?
3. **Check scope.** Are both true at different scales?
4. **Check definitions.** Are they measuring the same thing differently?
5. **Accept the tension.** Some contradictions are real and unresolvable. That's a finding in itself.

## Productive Contradictions

The most valuable research output is often: "The consensus says X, but the [[contrarian]] lens found Y, and here's why that matters."

When [[technical]] and [[contrarian]] disagree, pay special attention — one is looking at the data, the other is questioning the data's framing. Both are valid.

When [[economic]] and [[first-principles]] disagree, you've often found a place where incentives have distorted fundamental reality. That's gold.

## Log Unresolved Contradictions

If a contradiction can't be resolved, add it to [[research-log]] as an open question. It may be answerable with different data or in a future project.
